# Интернет-магазин "Ричмонд"

Профессионально сверстанный сайт интернет-магазина с админ-панелью для управления товарами и промо-блоками.

## Технологии

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **React**

## Установка

1. Установите зависимости:
```bash
npm install
```

2. Запустите dev-сервер:
```bash
npm run dev
```

3. Откройте [http://localhost:3000](http://localhost:3000) в браузере

## Структура проекта

- `/app` - страницы и API роуты
- `/components` - React компоненты
- `/components/admin` - компоненты админ-панели
- `/lib` - утилиты и работа с данными
- `/types` - TypeScript типы
- `/data` - локальное хранилище данных (JSON файлы)

## Админ-панель

Админ-панель доступна по адресу `/admin`:

- `/admin` - главная страница админ-панели
- `/admin/products` - управление товарами
- `/admin/promo` - управление промо-блоками

## Функционал

### Основной сайт
- Каталог товаров с фильтрацией по категориям
- Промо-баннеры на главной странице
- Подписка на рассылку
- Адаптивный дизайн

### Админ-панель
- Добавление, редактирование и удаление товаров
- Управление промо-блоками
- Управление порядком отображения промо-блоков
- Включение/отключение промо-блоков

## Данные

Данные хранятся в JSON файлах в директории `/data`:
- `products.json` - товары
- `promo.json` - промо-блоки

При первом запуске эти файлы создаются автоматически с тестовыми данными.

## Изображения

Вы можете использовать:
- Локальные изображения: поместите их в `/public/images/` и укажите путь `/images/имя-файла.jpg`
- Внешние URL: укажите полный URL изображения (например, `https://example.com/image.jpg`)

Next.js настроен для работы с внешними изображениями через `remotePatterns` в конфигурации.
